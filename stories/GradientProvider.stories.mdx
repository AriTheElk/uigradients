# Custom Gradient Presets

The `GradientProvider` allows you to pass a custom preset library into any `Gradient` component in your app. The recommended usage is to wrap as much of your app as you need in a `GradientProvider` and pass an object to the `presets` property. Like so:

```jsx
<GradientProvider
  presets={{ alihossein: { start: "#f7ff00", end: "#db36a4" } }}
>
  <Gradient preset="alihossein">
    This gradient is using the alihossein preset.
  </Gradient>
</GradientProvider>
```
